# Bachata S4 Android

The Android app is scaffolded with the `create-android@0.1.0` multi-module
template in `android/BachataS4`.

## Prerequisites

- JDK 17
- Android SDK 37
- Android NDK 29.0.14206865
- CMake
- Ninja
- Node.js 18 or newer

## Build

```sh
cd android/BachataS4
./gradlew test assembleDebug
```

Runtime assets are generated by `runtime/scripts/build-runtime.sh` and are not
stored in Git.
