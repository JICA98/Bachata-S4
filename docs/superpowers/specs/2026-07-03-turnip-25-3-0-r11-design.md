# Turnip 25.3.0 R11 Runtime Upgrade

## Goal

Replace bundled Winlator Turnip 25.0.0 with K11MCH1 Turnip 25.3.0 R11 while preserving Bachata S4's existing Vulkan loader and ICD paths.

## Source And Integrity

Use the exact release archive:

`https://github.com/K11MCH1/AdrenoToolsDrivers/releases/download/v25.3.0-rc.11/Turnip_v25.3.0_R11.zip`

Pin SHA-256 `0452bdd5e966ca94b58ba7c489db3648df6715d862e38cfa939ec62625293b40`. Packaging must reject a missing or modified archive. It must also validate `meta.json` identifies `Mesa Turnip Driver v25.3.0-R11`, minimum Android API 27, and library `vulkan.ad07xx.so`.

## Packaging

Fetch the ZIP through the existing locked runtime-input mechanism. Extract only the validated ARM64 driver. Install it as `host/libvulkan_freedreno.so`, keeping `host/vulkan/icd.d/freedreno_icd.json` and `VK_ICD_FILENAMES` unchanged. Do not modify the vendored Winlator component or delete its 25.0.0 archive; that preserves a straightforward rollback path.

## Failure Handling

Fail packaging before runtime generation when archive hash, metadata, or expected library differs. Never silently fall back to 25.0.0 because that would make device results ambiguous.

## Testing

Add a packaging regression test before implementation. It must initially fail against the 25.0.0 pin and then verify the RC11 URL, checksum, metadata contract, and packaged destination. Run runtime packaging, unit tests, clean APK assembly, install on the connected OnePlus Pad 2, and launch Gate 7. Acceptance requires Vulkan initialization, Gate 7 Running, sustained Sonic rendering, no device-lost/native crash, and correct full-screen aspect fit.

## Rollback

Reverting the upgrade commit restores the 25.0.0 package path and checksum. Existing source archives remain unchanged.
