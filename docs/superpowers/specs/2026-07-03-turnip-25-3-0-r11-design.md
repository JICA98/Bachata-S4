# Turnip 25.3.0 R11 Runtime Upgrade

## Goal

Add K11MCH1 Turnip 25.3.0 R11 as a selectable Android-native driver while preserving bundled Winlator Turnip 25.0.0 as the fallback.

## Source And Integrity

Use the exact release archive:

`https://github.com/K11MCH1/AdrenoToolsDrivers/releases/download/v25.3.0-rc.11/Turnip_v25.3.0_R11.zip`

Pin SHA-256 `0452bdd5e966ca94b58ba7c489db3648df6715d862e38cfa939ec62625293b40`. Packaging must reject a missing or modified archive. It must also validate `meta.json` identifies `Mesa Turnip Driver v25.3.0-R11`, minimum Android API 27, and library `vulkan.ad07xx.so`.

## Driver Bridge

Fetch the ZIP through the existing locked runtime-input mechanism. Extract only the validated ARM64 Bionic driver. Package it separately from the glibc runtime driver.

Build the libadrenotools sources already pinned in the Winlator source tree. Add an Android-native Vulkan proxy that initializes libadrenotools with the selected driver directory and exports `vkGetInstanceProcAddr`. Point Box64's native Vulkan wrapper at this proxy for RC11 sessions. Keep the existing glibc Vulkan loader, ICD, and Turnip 25.0.0 files unchanged for fallback sessions.

Driver choice is resolved before process launch and remains immutable for the session. Changing driver requires stopping and restarting emulation because an active Vulkan device cannot change its loader safely.

## Failure Handling

Fail packaging before runtime generation when archive hash, metadata, or expected library differs. Fail RC11 session preparation when the proxy cannot load the selected driver; never silently use the system driver because that would make device results ambiguous. Explicitly selecting the bundled 25.0.0 fallback continues to use the existing glibc path.

## Testing

Add packaging and driver-selection regression tests before implementation. Verify the RC11 URL, checksum, metadata contract, separate Bionic destination, native proxy selection, and explicit 25.0.0 fallback. Run runtime packaging, unit tests, clean APK assembly, install on the connected OnePlus Pad 2, and launch Gate 7. Acceptance requires RC11 Vulkan initialization, Gate 7 Running, sustained Sonic rendering, no device-lost/native crash, and correct full-screen aspect fit.

## Rollback

Selecting bundled Turnip 25.0.0 bypasses the native proxy and restores the known-working path. Existing source archives remain unchanged.
