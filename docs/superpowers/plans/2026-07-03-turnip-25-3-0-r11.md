# Turnip 25.3.0 R11 Native Proxy Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Load exact K11MCH1 Turnip 25.3.0 R11 through Android-native libadrenotools while retaining Winlator Turnip 25.0.0 fallback.

**Architecture:** Package RC11 as a Bionic driver. Build a native proxy backed by vendored libadrenotools; proxy exports `vkGetInstanceProcAddr` for Box64. Resolve driver selection before session launch and keep it fixed until restart.

**Tech Stack:** Android NDK/CMake, libadrenotools, Kotlin, Node.js ESM, Gradle, ADB, Box64, Vulkan.

## Global Constraints

- Use exact RC11 URL and SHA-256 from approved design.
- Validate metadata name, API 27, and `vulkan.ad07xx.so`.
- Never replace glibc `host/libvulkan_freedreno.so` with Bionic RC11.
- Keep 25.0.0 path available and explicit.
- Select driver before process launch; driver changes require session restart.
- Test only on OnePlus Pad 2 serial `7d6afed8`.
