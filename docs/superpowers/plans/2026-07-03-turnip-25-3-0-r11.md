# Turnip 25.3.0 R11 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Package and run the exact K11MCH1 Turnip 25.3.0 R11 release in the Android PS4 runtime.

**Architecture:** Treat the release ZIP as a locked runtime input. Validate archive checksum and `meta.json`, extract its ARM64 library, and copy it to the existing `host/libvulkan_freedreno.so` contract so loader configuration remains unchanged.

**Tech Stack:** Node.js ESM, JSON runtime locks, ZIP/unzip, Gradle, ADB, Box64, Vulkan.

## Global Constraints

- Source URL: `https://github.com/K11MCH1/AdrenoToolsDrivers/releases/download/v25.3.0-rc.11/Turnip_v25.3.0_R11.zip`.
- SHA-256: `0452bdd5e966ca94b58ba7c489db3648df6715d862e38cfa939ec62625293b40`.
- Require metadata name `Mesa Turnip Driver v25.3.0-R11`, `minApi` 27, and `libraryName` `vulkan.ad07xx.so`.
- Preserve destination `host/libvulkan_freedreno.so` and existing ICD path.
- Keep Winlator Turnip 25.0.0 source archive unchanged for rollback.
- Test only on connected OnePlus Pad 2.
